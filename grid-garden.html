Sobre o jogo

O Grid Garden é um jogo educativo interativo e gratuito projetado para ensinar propriedades de layout CSS Grid. O objetivo é escrever códigos CSS para regar cenouras, cultivar um jardim e posicionar elementos em um grid de 28 níveis, progredindo de comandos básicos para avançados. Jogar agora
