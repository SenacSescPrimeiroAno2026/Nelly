Flexbox Defense

Aprenda Flexbox Jogando

Um jogo divertido para praticar posicionamento com CSS Flexbox enquanto protege sua base dos inimigos.

Sobre o jogo

Flexbox Defense é um jogo que combina estratégia e aprendizado do modelo de layout CSS Flexbox. Nele, os jogadores precisam posicionar torres corretamente utilizando propriedades do Flexbox para impedir que os inimigos avancem até a base. Conforme você progride, os desafios ficam mais difíceis, tornando o aprendizado mais prático e divertido.
